# Car Price Prediction

## CodeAlpha Data Science Internship - Task 3

### Project Overview

This project predicts the selling price of used cars using machine learning techniques.

### Objective

To estimate the price of a used car based on various features.

### Features Used

* Year
* Present Price
* Kms Driven
* Fuel Type
* Seller Type
* Transmission
* Owner

### Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn

### Algorithm Used

Random Forest Regressor

### Project Workflow

1. Data Collection
2. Data Preprocessing
3. Train-Test Split
4. Model Training
5. Prediction
6. Model Evaluation

### Evaluation Metrics

* Mean Absolute Error (MAE)
* R2 Score

* car data.csv

* Car_Price_Prediction_Report.pdf
